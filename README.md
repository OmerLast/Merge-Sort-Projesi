# **Merge Sort Projesi**

## **Proje 2**
---

[16,21,11,8,12,22] -> Merge Sort

- Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
- Big-O gösterimini yazınız.

## **Cevaplar**

-        [16,21,11]   ve   [8,12,22] diye ayrılır.

-     [6] [21,11]              [8] [12,22]

- [6] [21] [11]                   [8] [12] [22]

-    [6,11] [21]              [8,12] [22]

-        [6,11,21]           [8,12,22]

-               [6,8,11,12,21,22]

- Merge sort türüne göre Big-O= O(nlogn)
